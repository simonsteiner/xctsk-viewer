About XCTSK Viewer

What is XCTSK Viewer?

XCTSK Viewer is a Python-based interactive visualization tool designed specifically for viewing and analyzing XCTSK (Cross Country Task) files. These files are commonly used in paragliding to define flight tasks, waypoints, and competition routes.

Features

  • Interactive file upload
  • Real-time task visualization
  • Waypoint analysis
  • Distance calculations
  • Modern web interface
  • Mobile-friendly design

Technology Stack

Backend
  • Python 3.8+
  • Flask web framework
  • XML parsing capabilities
Frontend
  • Bootstrap 5
  • JavaScript ES6+
  • Font Awesome icons

Getting Started

  1. Navigate to the Viewer page
  2. Upload your XCTSK file using drag-and-drop or the file browser
  3. View the interactive visualization of your task
  4. Analyze waypoints, distances, and flight patterns
