What is XCTSK Viewer?

XCTSK Viewer is a Python-based interactive visualization tool designed specifically for viewing and analyzing XCTSK (Cross Country Task) files. These files are commonly used in paragliding to define flight tasks, waypoints, and competition routes.

Features

  • Interactive file upload
  • Task visualization
  • Waypoint analysis
  • Distance calculations
  • Modern web interface
  • Mobile-friendly design

Technology Stack

Backend
  • Python 3.8+
  • Flask web framework
  • Powered by pyxctsk
  • XML parsing capabilities
Frontend
  • Bootstrap 5
  • JavaScript ES6+
  • Font Awesome icons
Analytics
  • Umami analytics
  • Privacy-focused tracking
  • Open-source solution
Hosting
  • Hosted on Fly.io
  • Global edge deployment
  • Docker containers

Powered by pyxctsk

This application is powered by pyxctsk, a Python implementation of XCTrack's task format for paragliding and hang gliding competitions. This library enables parsing, manipulation, and visualization of XCTrack task files used to define competition routes.

View pyxctsk on GitHub

Getting Started

  1. Navigate to the Viewer page
  2. Upload your XCTSK file using drag-and-drop or the file browser
  3. View the interactive visualization of your task
  4. Analyze waypoints, distances, and flight patterns
